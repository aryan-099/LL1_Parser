# LL(1) Parser

This is a basic LL(1) Parser built using Python. It takes a grammar and checks whether a given string is accepted or not based on LL(1) parsing technique.

## What it Does

- Takes user-defined grammar
- Builds the **FIRST** and **FOLLOW** sets
- Generates **Parse Table**
- Checks if the input string is valid or not
